# LearnFlow 

An AI-based adaptive learning platform that personalizes programming learning paths.

##  Features
- Generates quiz based on topic
- Assigns user level (Beginner / Intermediate / Advanced)
- Creates personalized roadmap
- Provides learning resources (videos + notes)

# Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- AI Integration: Ollama / OpenAI API

# 🚀 How it Works
1. User selects a topic  
2. System generates a quiz  
3. Based on performance → level is assigned  
4. Personalized roadmap is generated  
5. User gets learning resources
